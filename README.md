# SimpaTEXT

Расширение для Chrome с локальной LLM (Ollama) для упрощения текста, создания постов и тезисов.

## Функции

- 📖 Упрощение текста (3 уровня: лёгкое, среднее, максимальное)
- 📱 Генерация постов для соц.сетей
- 📝 Выделение тезисов
- 🌐 Адаптация всей страницы (с возвратом оригинала)
- 🌙 Тёмная и светлая темы
- ⌨️ Горячие клавиши (Ctrl+Shift+S, Ctrl+Shift+P, Ctrl+Shift+T)
- 🖱️ Контекстное меню (ПКМ на выделенном тексте)

## Установка

1. Установите [Ollama] (https://ollama.com)
2. Скачайте модель: в терминале напишите `ollama pull qwen2.5:7b`
3. Запустите сервер: также в терминале пропишите `ollama serve`
4. Загрузите расширение в Chrome (`chrome://extensions/` → Режим разработчика → Загрузить распакованное)

## Безопасность и приватность

- **Данные не покидают компьютер** — всё обрабатывается локально
- **Работа без интернета** — после установки модели
- **Без API-ключей** — невозможно скомпрометировать
- **Логирование отсутствует** — никто не хранит историю запросов

## Как пользоваться

- Выделите текст → ПКМ → SimpaTEXT → выберите режим
- Или используйте попап-окно (иконка расширения)
- Для всей страницы: нажмите «Адаптировать страницу» в попапе, выберите уровень адаптации и дожидайтесь результата

## Технологии

- Chrome Extension Manifest V3
- Ollama (локальный запуск LLM)
- Модель: Qwen 2.5 7B

## 💻 Минимальные требования к компьютеру

**Операционная система:** Windows 10 / macOS 11 / Linux (Ubuntu 20.04+)

**Процессор:** 4 ядра (Intel Core i5 / AMD Ryzen 3)

**Оперативная память:** от 8 ГБ (рекомендуется 16 ГБ)

**Свободное место:** около 6 ГБ (модель 4 ГБ + расширение)

**Видеокарта:** не обязательна, но ускоряет работу (CUDA / Metal)

**Интернет:** нужен только при первом скачивании модели (≈4 ГБ)

⚠️ На слабых ПК (4 ГБ RAM, старые процессоры) модель `qwen2.5:7b` может работать медленно или вылетать. В этом случае рекомендуем использовать модель `qwen2.5:1.5b` — она легче, но качество упрощения ниже.

## 🚧 Известные ограничения

**Локальная LLM.** Расширение не работает без установленного Ollama и запущенного сервера (`ollama serve`). Это сделано сознательно ради приватности.

**Высокое потребление памяти.** Модель `qwen2.5:7b` занимает до 8 ГБ оперативной памяти. На слабых компьютерах ответ может приходить за 10–30 секунд.

**Адаптация всей страницы.** Функция работает только на статичных сайтах (Википедия, блоги, статьи). На динамических страницах (React, Vue, SPA) текст может заменяться некорректно.

**CORS-ограничения в Chrome.** В обычном режиме расширение, в некоторых случаях, может не достучаться до Ollama. Для полностью корректной работы желательно запуск с флагом `--disable-web-security`. 
Вставьте это в cmd:
"C:\Program Files\Google\Chrome\Application\chrome.exe" --user-data-dir=C:\chrome-dev-data\ --disable-web-security


**Качество упрощения.** Модель иногда выдаёт английские слова (они фильтруются автоматически). При максимальном уровне упрощения теряются некоторые смысловые оттенки.

**Русский язык.** Модель обучена на русском, но может ошибаться с падежами и редкими словами.

## 🔧 Что делать, если что-то пошло не так

**Ошибка `Failed to fetch`** — Ollama не запущен. Откройте терминал и выполните `ollama serve`. Оставьте окно открытым.

**Ошибка `HTTP 403`** — проблема с доступом в обычном Chrome. Запустите браузер с флагом `--disable-web-security`.

**Модель не установлена** — скачайте её командой `ollama pull qwen2.5:7b`.

**Долгий ответ (больше 30 секунд)** — замените модель на более лёгкую: `ollama pull qwen2.5:1.5b` и поменяйте в коде "service_worker.js" (129 и 335 строки) в кавычках на qwen2.5:1.5b.
