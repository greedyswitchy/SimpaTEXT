# SimpaTEXT — локальный ИИ-помощник для чтения сложных текстов

Расширение для Chrome, которое упрощает технические тексты, расшифровывает термины и выделяет главные мысли.  
**Всё работает локально, без интернета, через вашу собственную LLM (Ollama).**

---

## Зачем это нужно?

Вы читаете документацию, ГОСТы, научные статьи или новости на технические темы?  
SimpaTEXT помогает быстрее понимать сложные тексты:

- **Упрощение** — переписывает сложные предложения простым языком (3 уровня сжатия)
- **Термины** — находит и объясняет сложные понятия и аббревиатуры
- **Тезисы** — выделяет 3–5 главных мыслей из любого текста

Расширение работает полностью **офлайн**. Ваши тексты никуда не отправляются.

---

## Что нужно для работы?

- **Операционная система:** Windows 7 / 10 / 11, macOS 11+, Linux
- **Процессор:** Intel Core i3 / AMD Ryzen 3 или новее (подойдёт любой современный)
- **Оперативная память:** от 4 ГБ (для лёгкой модели), лучше от 8 ГБ
- **Свободное место:** около 4–5 ГБ для модели

> На слабых компьютерах используйте модель `qwen2.5:1.5b`. На мощных — `qwen2.5:7b` для лучшего качества.

## Установка

В папке с расширением есть несколько файлов для быстрого старта:

- `SETUP_st.exe` — установщик, который автоматически настроит Ollama и модель
- `startPORT.bat` — запускает Ollama (если он не запущен)
- `stopPORT.bat` — останавливает Ollama (освобождает порт 11434)

### Быстрая установка

1. Запустите `SETUP_st.exe` от имени администратора
2. Следуйте инструкциям установщика
3. После завершения откройте Chrome и перейдите на `chrome://extensions/`
4. Включите «Режим разработчика»
5. Нажмите «Загрузить распакованное расширение» и выберите папку с файлами проекта

### Ручная установка (если установщик не подошёл)

1. Скачайте и установите [Ollama](https://ollama.com/download)
2. Откройте терминал и выполните: `ollama pull qwen2.5:1.5b`
3. Запустите Ollama: `ollama serve` (или используйте `startPORT.bat`)
4. Загрузите расширение в Chrome как распакованное

## Как пользоваться

**Через попап (иконка расширения)**
- Нажмите на иконку SimpaTEXT
- Выберите режим: Упрощение / Термины / Тезисы
- Вставьте текст в поле «Исходный текст»
- Нажмите «Инициализировать»

**Через контекстное меню (на любой странице)**
- Выделите текст мышкой
- Нажмите правую кнопку мыши
- Выберите `✨ SimpaTEXT` → нужное действие

**Горячие клавиши**
- `Ctrl + Shift + S` — упростить выделенный текст
- `Ctrl + Shift + P` — расшифровать термины
- `Ctrl + Shift + T` — выделить тезисы

**Адаптация всей страницы**
- Нажмите на иконку расширения
- Нажмите «Адаптировать страницу»
- Выберите уровень упрощения
- Появится плавающее окно с адаптированной версией

---

## Уровни упрощения

**Лёгкий** — замена сложных слов, сохранение всех деталей

**Средний** — удаление повторов, сохранение ключевых фактов и стиля

**Максимальный** — только суть, но с сохранением важных терминов

---

## Возможные проблемы и их решение

**❌ Ollama не запущен** — откройте терминал и выполните `ollama serve` или используйте `startPORT.bat`

**❌ Модель не найдена** — установите модель командой `ollama pull qwen2.5:1.5b`

**Долгая обработка** — используйте более лёгкую модель или текст до 2000 символов

**Не работает на сайте** — сайт блокирует расширения. Попробуйте на другом сайте или скопируйте текст в попап

**Пустой ответ** — текст слишком короткий (минимум 20 символов)
